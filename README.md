# Organizador de Campeonatos
O objetivo do projeto é criar uma plataforma para criação de campeonatos de Fifa. O site ajudaria as pessoas a se divertirem mais jogando, aumentaria a competitividade do jogo e os jogadores poderiam interagir fazendo novas amizades. Um objetivo também é premiar os jogadores que participam dos campeonatos, atraindo mais jogadores e deixando mais atrativo as partidas.
